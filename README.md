# Сервис по доставке продуктов

## Описание

Это небольшое приложение для управления заказами доставки продуктов.
### Основные возможности:
- Просматривать список заказов в таблице.
- Добавлять новые заказы через поля ввода.
- Отслеживать активность заказов по дням на графике.
- Логировать действия пользователя.
- Пользоваться меню с функцией выхода из приложения.

### Технические требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)
- PySide6 (для GUI интерфейса)
- sqlite(для работы с базой данных)
- matpotlib (для графического отображения данных)
### Установка зависимостей

Установите необходимые зависимости с помощью pip:

```bash
pip install PySide6
pip install matplotlib
pip install sqlite3
````
## Инструкции по запуску
-Скачайте или склонируйте файлы проекта
-Откройте терминал в папке с проектом
-Запустите главный файл приложения:
```bash
python Main.py
```
Для запуска pytest:
```bash
python pytests.py
```
